# kbDudeBrew Homebrew Editor

### Prerequisites
- Node.js

### Quick Install
- `git clone https://github.com/geoffboyd/kbDudeBrew.git`
- `cd kbBrewDude`
- `npm install`
- `npm start`
- From the same PC, open your browser and navigate to http://localhost:3001
